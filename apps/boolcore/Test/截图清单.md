## 截图技巧

1. **使用浏览器开发者工具**：F12 打开DevTools，可以调整窗口大小以获得清晰截图
2. **建议分辨率**：1280x720 或更高，便于展示细节
3. **截图工具**：
   - Linux: `gnome-screenshot` 或 `flameshot`
   - macOS: 原生截图 (Cmd+Shift+4 或 Cmd+Shift+5)
   - Windows: Snipping Tool 或 ShareX
4. **保存位置**：所有截图保存到 `/home/huihui/GolandProjects/BoolCore/screenshots/` 目录
