# BoolCore - 布尔函数密码学性质分析平台

## 项目概述

**BoolCore** 是一个先进的布尔函数密码学性质分析平台，专为密码学研究、教学和工程应用而设计。该平台集成了现代化的Web界面和高性能的Go后端，提供全面的布尔函数分析功能。

### 核心特性

- 🚀 **高性能计算**：基于Go语言的高效算法实现，支持位打包优化和并行计算
- 📊 **全面分析**：提供18+种密码学性质的完整分析
- 🎨 **现代化界面**：Vue3 + Vite构建的响应式Web界面
- 🔄 **多格式支持**：支持真值表、十六进制、整数、ANF四种输入格式
- 📁 **数据导出**：支持JSON、SageMath、Matlab等多种导出格式

#### 算法优化特性

**1. 位打包存储**
```go
// 使用uint64进行位打包，每个字存储64个布尔值
func uint64SliceFromTruthTable(tt []byte) []uint64 {
    n := len(tt)
    words := (n + 63) / 64
    out := make([]uint64, words)
    for i := 0; i < n; i++ {
        if tt[i] != 0 {
            out[i>>6] |= 1 << uint(i&63)
        }
    }
    return out
}
```

**2. 快速变换算法**
```go
// 快速Walsh-Hadamard变换，时间复杂度O(n·2^n)
func fwhtInplace(data []int64) {
    n := len(data)
    for step := 1; step < n; step <<= 1 {
        for i := 0; i < n; i += step << 1 {
            for j := 0; j < step; j++ {
                a := data[i+j]
                b := data[i+j+step]
                data[i+j] = a + b
                data[i+j+step] = a - b
            }
        }
    }
}
```
